##### 数学公式提取

发现明显局限。
部分数学符号可以保留：
```text
∑
≤
≥
```
但复杂公式结构会被破坏。如：
```text
L_t(y_t|C^S)
```
同时出现控制字符等异常 Unicode 字符：
```text
\u0013
```

结论：
> 公式内容部分保留，但公式排版和数学结构无法可靠恢复。
> 对于本项目而言影响有限，因为系统综述主要关注：理论/数据/方法名称/研究发现，而非公式本身。
